📌 Data Professional Survey Analysis (Power BI)



📊 Problem Statement

Analyzed survey data of data professionals to understand salary trends, job roles, programming preferences, and career challenges in the data industry.


🎯 Objective

To extract actionable insights from survey data and build an interactive dashboard to help aspiring data professionals understand industry trends and career paths.


🚀 Business Impact

Provides valuable insights for:
• Students choosing data careers
• Professionals planning skill upgrades
• Companies understanding employee satisfaction trends


📈 Key Insights / Results

• Data Scientists and Data Engineers earn the highest average salaries
• Python is the most preferred programming language among professionals
• Majority of respondents are from the United States and India
• Average age of professionals is ~30 years
• Work-life balance rating (~5.7/10) is higher than salary satisfaction (~4.3/10)
• Over 40% of respondents find it difficult to break into the data field


🗂️ Dataset

• Source: Data Professional Survey (public dataset)
• Size: 600+ survey responses
• Key Columns: Job Title, Salary, Country, Programming Language, Work-Life Balance, Career Difficulty


🛠️ Tools & Technologies

• Power BI
• Excel (Data Cleaning)
• DAX (Data Analysis Expressions)
• Data Visualization


🔍 Key Steps / Methodology

• Cleaned and transformed raw survey data in Excel
• Imported dataset into Power BI
• Created calculated measures using DAX
• Performed Exploratory Data Analysis (EDA)
• Built interactive dashboard with filters and slicers
• Visualized salary distribution by job roles
• Analyzed programming language preferences
• Evaluated work-life balance and salary satisfaction


📊 Visualizations / Dashboard

• Salary by Job Title (Bar Chart)
• Programming Language Preference (Column Chart)
• Country Distribution (Map/Bar Chart)
• Work-Life Balance vs Salary Satisfaction (KPI Cards)
• Career Difficulty Breakdown (Donut Chart)
